 משרה חמה
מיקום: ת"א והמרכז
תיאור המשרה:
החברה פיתחה את אחד המרקטפלייסים הגדולים בצפון אמריקה לחיפוש עבודה. לחברה מיליוני לקוחות והיא צומחת בקצב מהיר מאד כאשר מדובר בחברת B2B, B2C היות והלקוחות שלה הם גם המעסיקים וגם מחפשי העבודה ובכך האתגר אפילו גדול יותר ודורש חדשנות כל הזמן. לחברה אווירת סטארט-אפ ואחד מהערכים המשמעותיים זה התרבות הארגונית וה- Well Being של העובדים.

מהות התפקיד: חלק מצוות, פיתוח ב- React ועוד קצת ב- GO. כ- 70-80% פיתוח לצד ה- Client בסביבת AWS. ה- Stack כולל עבודה בין היתר עם Python, GO, Scala, Java, AWS, Kafka, DynamoDB, Elasticsearch, SQL ועוד.
דרישות המשרה:

-6 שנות ניסיון בפיתוח
-ניסיון ב- React

-ניסיון ב- Cloud + SQL
