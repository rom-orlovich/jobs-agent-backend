Fullstack Engineer לסטארט אפ בצמיחה

 משרה חמה
מיקום: ת"א והמרכז
תיאור המשרה:
מחפשים את הטאלנט/ית הבא/ה למשפחה!
אנו חברת סטארט אפ המפתחת מערכת סלולר B2B מאובטחת מבוססת ענן. הפלטפורמה מספקת קישוריות גלובלית לאינטרנט (IoT) וקישוריות מכונה למכונה (M2M) באמצעות ממשקי API רבי עוצמה.
השירותים כוללים ביקורת מלאה לכל אירועי הרשת עם זיהוי חריגות ואוטומציה של התראות.
מונים כ-20 עובדים וכעת בצמיחה, ממוקמים בתל אביב קו רכבת ועובדים במודל היברידי.
במסגרת התפקיד, פיתוח Fullstack ב-React ו-NodeJS.
לפרטים, שלח/י קורות חיים ונתעדכן!
דרישות המשרה:

- שנתיים ניסיון בפיתוח Frontend
- שנה ניסיון עם React
- ניסיון בפיתוח לצד ה-Server
